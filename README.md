# Simple Dead by Daylight Pip / Grade Tracker Overlay

A simple manual & visual tracker for your grade progress, from Ash 4 to Iridescent 1 with automatic saving.

##  Features
- **Auto-Save**: Progress is saved locally. It remembers your Pips even after closing the app.
- **Dynamic UI**: "Rank Up" checkmarks appears after completing a grade.
- **Global Hotkeys**: Control the tracker while in-game without tabbing out.

## Hotkeys
- `]` : **Add** +1 Pip
- `[` : **Subtract** -1 Pip

Can be changed by right clicking the software -> Settings
